# 🌤️ Weather Web App

A modern, responsive **Weather Web Application** that provides real-time weather data and hourly forecasts for any location worldwide. Built using **HTML, CSS, and JavaScript**, and powered by the Visual Crossing Weather API.

---

## 🚀 Features

* 🔍 Search weather by city or location
* 📍 Auto-detect user location (Geolocation API)
* 🌡️ Real-time temperature, wind speed, and precipitation
* ⏱️ 24-hour hourly forecast
* 🎨 Clean glassmorphism UI design
* 📱 Fully responsive (mobile-friendly)
* 🔄 Refresh weather data instantly
* 🌈 Dynamic weather icons using Feather Icons

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling (Glassmorphism + Responsive Design)
* **JavaScript (ES6)** – Logic and API handling
* **Visual Crossing Weather API** – Weather data provider
* **Feather Icons** – Weather icons

## ⚙️ Installation & Setup

1. **Clone the repository**

```bash
git clone https://github.com/endoalex111-hash/Weather-App.git
```

2. **Navigate into the project folder**

```bash
cd Weather-App
```

3. **Open the project**

* Simply open `index.html` in your browser
  OR
* Use Live Server (recommended)

---

## 🔑 API Key Setup

This project uses the **Visual Crossing Weather API**.

1. Get your free API key from:
   👉 https://www.visualcrossing.com/

2. Replace the API key in `script.js`:

```js
const apiKey = "YOUR_API_KEY_HERE";
```

---

## ⚠️ Important Note

> This project currently uses a **client-side API key**, which is visible in the browser.
> For production use, it is recommended to:

* Move API calls to a backend (Node.js / Express)
* Store API keys in environment variables
